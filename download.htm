Okay, here's the deal... I've zipped up all the files for this version of Pop WILL Sample Itself into a single binary file that you can load into your own DOS or UNIX PC. So what? you say? Well, for one thing, you can definitely access the files faster in local mode than over a dialup connection... For another thing, you may be able to save yerself some connect-time charges.

Here's the details:
You need PKUNZIP Version 2.x and about 2 meg of free space on your hard drive. The zip file is about 1.3MB, and contians over 100 files.
  1. Download PWSI
  2. Create a directory on your PC to hold all the pwsi files. ("PWSI" is a nice choice.)
  3. Run PKUNZIP (V2.x) like this: PKUNZIP PWSIV402.ZIP \PWSI\ *.*
  4. Once the files are decompressed into your directory, access PWSI.HTM with your web browser. (Note that there are a few links that point outside the local set of files. You may get errors trying to access these, but hitting STOP should take care of these.
